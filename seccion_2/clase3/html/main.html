        El DOM es la estructura de objetos

    y se puede alterar mediante Javascript para cambiar
    dinámicamente los contenidos y aspecto de la página.

  • Id y ClassName

    Codigo 
        var elementoConId = document.getElementById("idDelElemento");

    var elementoConClassName = document.getElementByClassName("classNameDelElemento");
